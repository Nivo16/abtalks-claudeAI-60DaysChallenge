# Day 6 – ATS Resume Optimization Challenge

## Objective
Optimize my resume using AI and improve ATS compatibility while keeping all information truthful.

## Original Resume
- Uploaded my original CV.
- ATS score before optimization: 68/100.

## Optimized Resume
- ATS score after optimization: 89/100.
- Improved keyword matching.
- Better structure for ATS parsing.
- Added dedicated Projects section.
- Removed duplicate skills.

## Key Learnings

1. ATS systems prefer standard section headings.
2. Strong action verbs improve readability.
3. Projects deserve their own section.
4. Avoid repeating the same skills multiple times.
5. Include relevant technical keywords naturally.
6. Keep formatting simple and ATS-friendly.
